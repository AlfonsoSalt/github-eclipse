Análisis del problema
El problema es el siguiente:
The following program randomly generates an integer greater than or equal to 0 and less than 100. The program then prompts the user to guess the number. If the user guesses the number correctly, the program outputs an appropriate message. Otherwise, the program checks whether the guessed number is less than the random number. If the guessed number is less than the random the number generated by the program, the program outputs the message, ‘‘Your guess is lower than the number’’; otherwise, the program outputs the message, ‘‘Your guess is higher than the number’’. The program then prompts the user to enter another number. The user is prompted to guess the random number until the user enters the correct number.
The program uses the method random of the class Math to generate a random number.

Las instrucciones son sencillas y existen 3 posibilidades, el numero fue correcto, menor, o mayor.
La manera más simple por el poco numero de casos son 3 if/ifelse.

Para manejar los numeros realmente solo se ocupan 2, el numero ingresado "res" y el numero generado "num1"
Como implementacion personal agregue cun contador "i" del numero de intentos

Estas variables serán :
Entradas: num1 (el numero que ingresa el usuario)
Salidas: res (el numero a adivinar) y i (numero de intentos)

Los tres serán integers debido a que se piden números simples
    int num1=0;
		int res=0;
		int i=0;

Posteriormente para generar el numero aleatorio se usará la librería math, y para limitarlo entre numeros del 0 al 100 se usarán las operaciones multiplicación y posteriormente módulo
  num1= (int) ((Math.random()*1000)%100);	

solo falta generar un bucle con la condicion que termine hasta que sean iguales, con un ciclo do while para asegurarnos mostrar los mensajes correspondientes encapsulando las 3 opciones

Este es el algoritmo final:
//////////////////////////////////////////////////////////////////////////////////////
package act1_5;
import java.util.Scanner;
//testest
public class GuessANumber {

	public static void main(String[] args) {
		
		int num1=0;
		int res=0;
		int i=0;

		
		System.out.println("Number guesser game ");
		System.out.println("Try to guess a number and find the correct number with my hints ");
		num1= (int) ((Math.random()*1000)%100);	
		System.out.println("Enter your number an press enter: ");
		
		do{

				Scanner input= new Scanner(System.in);
				res=input.nextInt();
			
			if(res==num1) {
				System.out.println("Congratulations! the number was "+num1);
				System.out.println("You tried "+i +" times.");
			}
			else if(res<num1) {
				System.out.println("Your number is lower than the answer ");
				i++;
			}
			else if(res>num1) {
				System.out.println("Your number is higher than the answer ");
				i++;
			}
		
		}while(num1!=res) ;
}
	
}
